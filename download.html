Installation
1

Lancez l'installateur

Double-cliquez sur le fichier ArenaMudir_0.1.0_x64-setup.exe téléchargé. L'assistant d'installation démarre.

2

Entrez votre clé de licence

Au premier démarrage, saisissez la clé reçue par message : AM-XXXX-XXXX-XXXX

3

Configurez vos TVs

Suivez l'assistant : nombre de stations, types, tarifs, et adresses IP des téléviseurs. C'est prêt !

⚠

Avertissement Windows SmartScreen

Lors du premier lancement, Windows peut afficher un avertissement de sécurité. Cliquez sur « Informations complémentaires » puis « Exécuter quand même ». C'est normal pour les applications récentes — la signature numérique sera ajoutée prochainement.
